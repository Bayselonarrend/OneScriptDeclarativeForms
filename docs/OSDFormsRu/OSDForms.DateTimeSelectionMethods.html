ВыборДатыВремени (DateTimeSelection) Методы

Методы

Выбрать (Select) Выбирает содержимое текстового поля даты и времени.
ПрокрутитьДо (ScrollIntoView) Прокручивает элемент в видимую область окна браузера.
СделатьНажатие (MakeClick) Имитация щелчка мыши по элементу.
СнятьФокус (Blur) Удаляет фокус с элемента.
Удалить (Remove) Удаляет элемент.
Фокус (Focus) Устанавливает фокус на элементе.
ШагВверх (StepUp) Увеличивает значение поля даты и времени на указанное число.
ШагВниз (StepDown) Уменьшает значение поля даты и времени на указанное число.
