ДекларативныеФормы.ПриПолученииСвойства (DeclarativeForms.GettingProperty) Событие

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код, указывающий на обработчик события при получении свойства объекта формы методом ДекларативныеФормы.ПолучитьСвойство (DeclarativeForms.GetObjectProperty).

Примечание

Пример

ДФ.ПриПолученииСвойства = ДФ.Действие(ЭтотОбъект, "ДФ_ПриПолученииСвойства");
