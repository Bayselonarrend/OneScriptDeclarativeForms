Четырёхугольник (Tetragon) Конструктор

Синтаксис

Четырёхугольник(<Икс1>, <Игрек1>, <Икс2>, <Игрек2>)

Параметры

Икс1 (обязательный)
Тип: Число.
Координата по оси икс нижнего левого угла прямоугольника.
Игрек1 (необязательный)
Тип: Число.
Координата по оси игрек нижнего левого угла прямоугольника.
Икс2 (необязательный)
Тип: Число.
Координата по оси икс верхнего правого угла прямоугольника.
Игрек2 (необязательный)
Тип: Число.
Координата по оси игрек верхнего правого угла прямоугольника.

Описание

Создаёт новый экземпляр класса Четырёхугольник (Tetragon).
