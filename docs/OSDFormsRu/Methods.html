Библиотека DeclarativeForms Методы

Методы

Методы (Methods) Описание
ВвестиГлобальноеСвойство (InjectGlobalProperty) (ДекларативныеФормы) Вводит глобальное свойство в область видимости загружаемых скриптов.
Вставить (Insert) (Меню) Вставляет элемент меню в указанную позицию меню. Индекс начинается с 0.
Выполнить (Execute) (ДекларативныеФормы) Вызывает заданный метод.
Добавить (Append) (Меню) Добавляет элемент в конец меню.
ЗагрузитьСценарии (LoadScripts) (ДекларативныеФормы) Находит файлы сценария по указанному пути, вставляет их в структуру, которую подключает к текущей системе типов.
Закрыть (Close) (Диалог) Закрывает диалоговое окно.
НайтиЭлемент (FindElement) (ДекларативныеФормы) Возвращает элемент управления формы, найденный по его имени.
Открыть (Open) (Форма) Открывает форму.
Показать (Popup) (Меню) Показывает контекстное меню по привязке (x, y) в текущем окне.
Показать (Show) (Диалог) Показывает диалог.
СообщитьОЗапускеФормы (InformLaunchForm) (ДекларативныеФормы) После открытия форма отправит POST запрос TCP серверу с сообщением - Форма запустилась. Отображение этого сообщения в консоли подтверждает открытие формы и установление связи со сценарием.
Тест (Test) (ДекларативныеФормы) Производит проверку равенства свойств элементов запущенной формы свойствам, заданным до запуска формы.
Удалить (Remove) (Меню) Удаляет пункт из меню.
УдалитьПоИндексу (RemoveAt) (Меню) Удаляет пункт из меню.
Цвет (Color) (ДекларативныеФормы) Возвращает значение цвета в виде строки.
