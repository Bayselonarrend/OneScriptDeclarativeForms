Библиотека DeclarativeForms События

События

События (Events) Описание
ДвойноеНажатие (DoubleClick) (Элемент) Возвращает или задает код, когда элемент дважды щелкнут.
Загружена (Loaded) (Форма) Возвращает или задает код, когда форма полностью загружена.
Нажатие (Click) (Элемент) Возвращает или задает код для выполнения при нажатии элемента управления.
ПриОтпусканииМыши (MouseUp) (Элемент) Возвращает или задает код для выполнения при отпускании кнопки мыши, когда указатель мыши находится на элементе.
