Холст.РисоватьГрафику (Canvas.DrawImage) Метод

Синтаксис

РисоватьГрафику(<Изображение>, <Икс>, <Игрек>)

РисоватьГрафику(<Изображение>, <Икс>, <Игрек>, <Ширина>, <Высота>)

РисоватьГрафику(<Изображение>, <Икс>, <Игрек>, <Ширина>, <Высота>, <ИксОбрезки>, <ИгрекОбрезки>, <ШиринаОбрезки>, <ВысотаОбрезки>)

Параметры

Изображение (обязательный)
Тип: Произвольный.
Изображение, холст или видео, которое будет использоваться.
Икс (обязательный)
Тип: Число.
x координата размещения изображения на холсте.
Игрек (обязательный)
Тип: Число.
y координата размещения изображения на холсте.
Ширина (необязательный)
Тип: Число.
Ширина изображения.
Высота (необязательный)
Тип: Число.
Высота изображения.
ИксОбрезки (необязательный)
Тип: Число.
x корордината начала обрезки.
ИгрекОбрезки (необязательный)
Тип: Число.
y координата начала обрезки.
ШиринаОбрезки (необязательный)
Тип: Число.
Ширина обрезанного изображения.
ВысотаОбрезки (необязательный)
Тип: Число.
Высота обрезанного изображения.

Возвращаемое значение

Описание

Рисует изображение, холст или видео на холсте.
