Действие.Параметр (Action.Parameter) Свойство

Использование

Чтение и запись.

Значение

Тип: Произвольный.

Возвращает или задает объект, заданный при установке обработчика события, для передачи в обработчик события.

Примечание

Пример

Кнопка1.Нажатие = ДФ.Действие(ЭтотОбъект, "Кнопка1_Нажатие", Структура1);
