РазрывСтраницыПосле (PageBreakAfter) Перечисление

Описание

Указывает поведение разрыва страницы после элемента.

Члены

Имя члена Описание Значение
Авто (Auto) Разрыв страницы вставляется при необходимости после элемента. Это значение по умолчанию. auto
Всегда (Always) Разрыв страницы после элемента вставляется всегда. always
Лево (Left) Вставляется один или два разрыва страницы после элемента, чтобы следующая страница считалась левой. left
Право (Right) Вставляется один или два разрыва страницы после элемента, чтобы следующая страница считалась правой. right
Предотвратить (Avoid) Разрыв страницы после элемента предотвращается. avoid
ПустаяСтрока (EmptyString) Разрыв страницы после элемента не вставляется. empty string
