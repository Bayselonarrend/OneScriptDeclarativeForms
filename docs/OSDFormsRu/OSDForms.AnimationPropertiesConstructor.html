СвойстваАнимации (AnimationProperties) Конструктор

Синтаксис

СвойстваАнимации(<ИмяАнимации>, <ДлительностьАнимации>, <ФункцияСинхронизации>, <Задержка>, <КоличествоПовторов>, <Направление>, <Заливка>, <Состояние>)

Параметры

ИмяАнимации (необязательный)
Тип: Строка.
Имя набора ключевых кадров анимации.
ДлительностьАнимации (необязательный)
Тип: Число.
количество миллисекунд длительности одного цикла анимации.
ФункцияСинхронизации (необязательный)
Тип: ФункцияСинхронизации (AnimationTimingFunction).
Определяет кривую скорости анимации.
Задержка (необязательный)
Тип: Число.
Задержка перед началом анимации в миллисекундах.
КоличествоПовторов (необязательный)
Тип: Число.
Количество повторений анимации. Значение -1 соответствует бесконечному воспроизведению анимации.
Направление (необязательный)
Тип: НаправлениеАнимации (AnimationDirection).
Определяет значение, должна ли анимация воспроизводиться в прямом или обратном порядке.
Заливка (необязательный)
Тип: ЗаливкаАнимации (AnimationFillMode).
Определяет значения стилей ключевых кадров которые применяются до или после анимации.
Состояние (необязательный)
Тип: Состояние (AnimationPlayState).
Определяет запущена или приостановлена ​​анимация.

Описание

Создаёт новый экземпляр класса СвойстваАнимации (AnimationProperties).
