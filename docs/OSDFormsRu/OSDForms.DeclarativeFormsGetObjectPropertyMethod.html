ДекларативныеФормы.ПолучитьСвойство (DeclarativeForms.GetObjectProperty) Метод

Синтаксис

ПолучитьСвойство(<Элемент>, <Свойство>)

Параметры

Элемент (обязательный)
Тип: Произвольный.
Элемент формы свойство которого необходимо получить.
Свойство (обязательный)
Тип: Строка.
Имя получаемого свойства.

Возвращаемое значение

Описание

Получает для указанного элемента указанное свойство, которое будет доступно для обработки в обработчике события ДекларативныеФормы.ПриПолученииСвойства (DeclarativeForms.GettingProperty).

Примечание

Метод можно вызывать только после загрузки формы. То есть нельзя вызывать до вызова метода Форма.Открыть (Form.Open). Чтобы обойти это ограничение можно вызвать метод в пределах обработчика события формы Форма.Загружена (Form.Loaded). Текущее значение указанного в параметрах свойства будет доступно в обработчике события ДекларативныеФормы.ПриПолученииСвойства (DeclarativeForms.GettingProperty).

Пример

ДФ.ПолучитьСвойство(Абзац1, "Направление");
