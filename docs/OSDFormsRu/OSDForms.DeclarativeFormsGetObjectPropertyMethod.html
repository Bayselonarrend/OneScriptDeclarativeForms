ДекларативныеФормы.ПолучитьСвойство (DeclarativeForms.GetObjectProperty) Метод

Синтаксис

ПолучитьСвойство(<Элемент>, <Свойство>)

Параметры

Элемент (обязательный)
Тип: Произвольный.
Элемент формы свойство которого необходимо получить.
Свойство (обязательный)
Тип: Строка.
Имя получаемого свойства.

Возвращаемое значение

Описание

Получает для указанного элемента указанное свойство, которое будет доступно для обработки в обработчике события ДекларативныеФормы.ПриПолученииСвойства (DeclarativeForms.GettingProperty).
