Библиотека DeclarativeForms Классы

Классы

Абзац (Paragraph) Представляет элемент HTML <p>.
АргументыСобытия (EventArgs) Представляет класс, содержащий данные событий.
Действие (Action) Задает обработчик события.
ДекларативныеФормы (DeclarativeForms) Объект ДекларативныеФормы (DeclarativeForms) содержит свойства и методы для создания и управления декларативными формами.
Диалог (Dialog) Представляет элемент HTML <dialog> (всплывающее окно или диалог).
Кнопка (Button) Представляет элемент HTML <button>.
Меню (Menu) Представляет структуру меню формы.
Форма (Form) Представляет окно, которое составляет пользовательский интерфейс приложения.
Элемент (Element) Представляет базовый класс от которого наследуются элементы HTML.
ЭлементМеню (MenuItem) Представляет отдельный элемент меню.
