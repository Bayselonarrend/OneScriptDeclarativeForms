ДекларативныеФормы.Тест (DeclarativeForms.Test) Метод

Синтаксис

Тест(<Элемент>, <ИмяСвойства>)

Параметры

Элемент (обязательный)
Тип: Произвольный.
Элемент, свойство которого проверяется.
ИмяСвойства (обязательный)
Тип: Строка.
Русскоязычное имя свойства элемента.

Возвращаемое значение

Описание

Производит проверку равенства свойств элементов запущенной формы свойствам, заданным до запуска формы.

Примечание

Успешное тестирование подтверждает запуск сервера, запуск формы и отправку сообщения сценарию через сервер.

Если в сообщениях сценария не будет содержаться три восклицательных знака !!!, значит тест прошел успешно.

Пример

ДФ.Тест(Кнопка1, "Высота");

ПодключитьВнешнююКомпоненту("C:\444\111\DeclarativeForms\DeclarativeForms\bin\Debug\DeclarativeForms.dll");
ДФ = Новый ДекларативныеФормы();
Форма1 = ДФ.Форма;

Абзац1 = ДФ.Абзац();
Абзац1.Родитель = Форма1;
Абзац1.Высота = 200;
Абзац1.Текст = "Абзац1";
Абзац1.ЦветФона = ДФ.Цвет("Желтый");

Кнопка1 = ДФ.Кнопка();
Кнопка1.Родитель = Абзац1;
Кнопка1.Ширина = 250;
Кнопка1.Высота = 35;
Кнопка1.Текст = "Кнопа";
Кнопка1.ЦветФона = ДФ.Цвет("Аквамариновый");
Кнопка1.Нажатие = ДФ.Действие(ЭтотОбъект, "Кнопка1_Нажатие");

ДФ.Тест(Кнопка1, "Ширина");
ДФ.Тест(Кнопка1, "Высота");
ДФ.Тест(Кнопка1, "Текст");
ДФ.Тест(Кнопка1, "ЦветФона");
ДФ.Тест(Кнопка1, "Родитель");

Форма1.Открыть();
